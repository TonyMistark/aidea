# App Store 元数据与素材准备

## 一、App 名称与副标题

| 字段 | 字数 | 建议 |
|---|---|---|
| **App 名称** | 30 字符 | `Aidear - AI 写作助手` |
| **副标题** | 30 字符 | `碎片想法，瞬间变成好文章` |

> 名称里带"AI 写作"对搜索排名有帮助，但不要堆砌关键词，Apple 会拒。

## 二、描述文案

### 促销文本（170 字符，可随时改，无需发版）

```
把你的零散想法交给 AI，30 秒生成一篇结构完整、排版精美的公众号文章。支持微信格式一键复制，写文章从未如此简单。
```

### App 描述（4000 字符，审核后修改需发版）

```markdown
【Aidear — 让每个人都成为表达者】

你是否也有过这些时刻？
• 脑子里有很多想法，但不知道怎么把它们组织成一篇好文章
• 想写公众号但总是卡在开头，对着空白文档发呆
• 工作太忙，没时间打磨文字，但又不想放弃表达
• 有观点想分享，但不知道怎么写得让别人爱看

Aidear 就是为你而生的。

▎怎么用？

1. 把你想说的话写下来（哪怕只是几个关键词、一段碎碎念）
2. AI 自动帮你扩展成一篇结构完整、可读性强的文章
3. 一键复制为微信格式，直接粘贴到公众号编辑器发布

就这么简单。你不需要会写作，你只需要有想法。

▎为什么选择 Aidear？

• 真正的公众号写作思维
AI 不是简单地扩写，而是按照公众号文章的写作逻辑来组织内容——开篇抓人、论据有力、结尾有延伸。生成的每篇文章都包含标题、摘要、正文、封面图提示词。

• 专为手机阅读优化
自动应用微信排版的视觉规则：标题用绿色强调、核心观点用粗体标出、引用用灰色区块、表格自动适配手机屏幕宽度。生成的排版在手机上看起来就是舒服。

• 一键复制微信格式
复制按钮复制的不是纯文本，而是带完整排版的 HTML。直接粘贴到微信公众平台编辑器，样式一个不丢。

• 隐私优先
你的想法只有你自己知道。我们不会存储你的输入内容，不会用你的数据训练模型。

▎适合谁用？

• 公众号创作者 — 保持日更频率的秘密武器
• 职场人士 — 工作报告、复盘总结、团队分享一气呵成
• 学生 — 读书笔记、学习心得轻松写
• 想开始写作但不知道从哪下手的人 — 这就是你的第一步

下载 Aidear，把你的想法变成文章。第一个灵感，现在就开始。
```

## 三、关键词

100 字符以内，用英文逗号分隔，不用空格。

```
AI写作,公众号助手,文章生成,写作助手,AI创作,文案生成,微信排版,智能写作,内容创作,公众号排版,AI写文章,写作工具,文章排版,AI润色,公众号编辑
```

**选词思路**：
- 核心功能词：AI写作、文章生成、公众号助手
- 长尾词：公众号排版、微信排版、AI润色
- 竞品词：智能写作、写作助手（别人搜这些词也能找到你）

## 四、App 图标

### 规格要求
- **App Store 展示**：1024×1024 px，PNG，无透明，sRGB
- **不要**在图标上加"Beta"、"Preview"字样（会被拒）
- **不要**用圆角（系统自动加）

### 设计建议

Aidear 的含义是 AI + Idea，建议设计思路：
- 主图形：一个发光的灯泡（代表想法）+ 一支笔（代表写作），或者
- 简洁的文字图标：一个"写"字的艺术变形，或者
- 抽象图形：一个由文字组成的对话气泡

色调：蓝紫渐变（呼应 AI 感）+ 暖白底色

### 制作工具
- **Figma**（免费网页版）→ 1024×1024 画布，导出 PNG
- **Splash（splash.iysa.africa）** → AI 生成 App 图标
- 设计师外包：大概 300-800 元

> 图标是用户对 App 的第一印象，值得投入。如果自己不会设计，建议花几百块找设计师画一个。

## 五、截图

### 必需尺寸

| 设备 | 分辨率 | 你的测试机 |
|---|---|---|
| 6.7" iPhone | 1290×2796 | iPhone 17 Pro Max |
| 6.1" iPhone | 1179×2556 | iPhone 17e / Air |

> App Store Connect 要求至少上传 6.7" 的截图，系统会自动缩放到其他尺寸。但手动准备 6.1" 的会更精确。

### 截图数量

每个尺寸至少 3 张，最多 10 张。**前 3 张最重要**，它们在搜索结果中直接可见。

### 截图内容规划

| 序号 | 画面 | 卖点 |
|---|---|---|
| 1 | 输入界面 + 几句输入示例 | "把想法说出来，剩下的交给 AI" |
| 2 | 生成结果的文章预览（用好看的样本文章） | "结构完整、排版精美、即写即看" |
| 3 | 微信格式预览 + 复制按钮 | "一键复制，直接粘贴到微信" |
| 4（可选） | 设置页面 | "支持自定义 AI 模型" |
| 5（可选） | 多篇文章展示 | "文章、总结、复盘，都能写" |

### 截图顶部文字

在截图上叠加大号文案（用 Keynote/Canva/Figma 都可以做）：

```
图 1："有想法，就能写文章"
图 2："AI 自动扩展，结构清晰"
图 3："微信格式，一键复制"
```

### 制作工具
- **Apple Frames**（Figma 插件）→ 把截图放进 iPhone 框架中
- **AppMockup**（app-mockup.com）→ 网页版截图美化
- **Screen Studio**（付费 macOS App）→ 旋转展示
- **Keynote** → 最基础，叠文字 + 导出图片

## 六、技术支持 URL

至少要有一个网页。最简单的方案：

1. 在 GitHub 创建一个公开仓库 `aidear-support`
2. 启用 GitHub Pages
3. 写一个简单的 index.html，包含：
   - App 名称和简介
   - 联系方式（邮箱）
   - FAQ

或者在 Notion 创建一个公开页面也可以。

## 七、审核备注

提交审核时，在"App 审核信息"部分填写：

```
【登录信息】（如果有了 Sign in with Apple 就不需要测试账号）
本 App 使用 Sign in with Apple 登录，审核团队用任何 Apple ID 即可登入。

【AI 功能说明】
本 App 的核心功能是将用户输入的文字发送到 AI 服务进行处理，生成结构化的文章。
AI 服务使用 OpenAI API，不存储用户数据。
生成的内容不包含色情、暴力或其他不当内容。

【内容过滤】
输入内容经过 OpenAI Moderation API 过滤。
```
