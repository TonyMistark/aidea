# Aidear Logo 设计文档

## 概览

Aidear = AI + Idea。Logo 用两个字母 **I** 和 **A** 直接传达产品核心：AI 将碎片想法转化为结构化文章。

## 设计决策

### 核心概念：I + A

| 字母 | 含义 | 视觉表现 |
|---|---|---|
| **I** | Idea（想法、灵感） | 琥珀色闪电光柱 + 顶部火花，暗示灵感的迸发 |
| **A** | AI（人工智能） | 蓝色几何三角框架，暗示结构化和理性 |

布局：**A 在左，I 在右**。从左到右的阅读顺序自然形成 "AI 处理 Idea" 的叙事。

### 视觉元素

**背景**：深蓝近黑 `#121424` 圆角正方形，低调不抢主体。

**I（Idea）**：
- 琥珀色 `#F8B448` 三段折线（闪电形），象征灵感的跳跃和能量
- 顶部四角星 `#FFE68C` 配三个小光点，强化火花感
- 周围柔和的暖色光晕（高斯模糊矩形），增加温暖感

**A（AI）**：
- 蓝色 `#488ED7` 锐角三角形 + 水平横杠，干净的几何形状
- 与 I 的有机曲线形成对比——理性框架 vs 感性灵感

**连接线**：顶部一条细微的蓝色弧线，暗示 AI 与 Idea 之间的连接和能量流动。

### 颜色系统

| 用途 | 颜色 | 十六进制 |
|---|---|---|
| 背景 | 深蓝近黑 | `#121424` |
| A（AI）| 科技蓝 | `#488ED7` |
| I（Idea）| 暖琥珀 | `#F8B448` |
| 火花 | 亮金黄 | `#FFE68C` |
| 连接线 | 柔蓝 | `#6E9BF0` |
| 光晕 | 琥珀低透 | `#E6A541` 100/255 |

### 构图参数

- 画布：1024×1024px
- 背景圆角：120px（对应 iOS App Icon 裁切）
- 边距：40px
- A 中心 x = 412（略偏左）
- I 中心 x = 652（偏右，与 A 保持视觉平衡）
- 两者顶部对齐，高度分别为 500px / 480px

## 技术实现

### 源文件

- 生成脚本：`docs/logo/gen-logo.py`（Python + Pillow）
- 输出：`aidear/Assets.xcassets/AppIcon.appiconset/icon-ia-final-flipped.png`
- 应用图标：`aidear/Assets.xcassets/AppIcon.appiconset/AppIcon.png`（直接复制）
- 配置：`Contents.json` 中 `filename` 字段指向 `AppIcon.png`

### iOS 适配

- 目标平台：iOS 17+
- 仅需 `1024×1024` 单一尺寸（Universal，iOS platform）
- Xcode 自动处理各设备尺寸的缩放和圆角裁切

## 未来调整方向

- **I 位置**：修改 `i_cx = SIZE//2 + OFFSET` 中的 OFFSET 值（当前 140）

- **颜色调整**：替换十六进制值即可
- **闪电段数**：增删 `pts` 数组中的坐标点
- **火花样式**：修改星形多边形的半径和点数
